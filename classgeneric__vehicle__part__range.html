Detailed Description

template<typename range_type>
class generic_vehicle_part_range< range_type >

The generic range, it misses the bool contained(size_t) function that is required by the iterator class.

You need to derive from it and implement that function. It uses the curiously recurring template pattern (CRTP), so use your derived range class as range_type.
