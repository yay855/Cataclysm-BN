157/**
158 * A duration defined as a number of specific time units.
159 * Currently it stores the number (as integer) of turns.
160 * Note: currently variable season length is ignored by this class (N turns are
161 * always N turns) and there is no way to create an instance from units larger
162 * than days (e.g. seasons or years) and no way to convert a duration into those
163 * units directly. (You can still use @ref calendar to do this.)
164 *
165 * Operators for time points and time duration are defined as one may expect them:
166 * -duration ==> duration (inverse)
167 * point - point ==> duration (duration between to points in time)
168 * point + duration ==> point (the revers of above)
169 * point - duration ==> point (same as: point + -duration)
170 * duration + duration ==> duration
171 * duration - duration ==> duration (same as: duration + -duration)
172 * duration * scalar ==> duration (simple scaling yields a double with precise value)
173 * scalar * duration ==> duration (same as above)
174 * duration / duration ==> scalar (revers of above)
175 * duration / scalar ==> duration (same as: duration * 1/scalar)
176 * duration % duration ==> duration ("remainder" of duration / some integer)
177 * Also shortcuts: += and -= and *= and /=
178 */
