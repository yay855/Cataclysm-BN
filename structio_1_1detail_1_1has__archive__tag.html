Detailed Description

template<class T, class E = void>
struct io::detail::has_archive_tag< T, E >

Implementation for classes that don't have an archive_type_tag defined.

They use the normal JsonSerializer / JsonDeserializer interface, which is handled directly by the Json classes. Therefor the functions here simply forward to those.
