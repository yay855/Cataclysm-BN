Detailed Description

Standard arguments for finding overmap terrain.

Parameters
originLocation of search
typesvector of Terrain type/matching rule to use to find the type
search_rangeThe maximum search distance. If 0, OMAPX is used.
min_distanceMatches within min_distance are ignored.
must_seeIf true, only terrain seen by the player should be searched.
cant_seeIf true, only terrain not seen by the player should be searched
existing_overmaps_onlyIf true, will restrict searches to existing overmaps only. This is particularly useful if we want to attempt to add a missing overmap special to an existing overmap rather than creating many overmaps in an attempt to find it.
om_specialIf set, the terrain must be part of the specified overmap special.
popupIf set, the popup will be periodically updated to indicate ongoing search.
