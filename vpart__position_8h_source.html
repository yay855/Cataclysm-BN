23/**
24 * Reference to a position (a point) of the @ref vehicle.
25 * It does not refer to a specific vehicle part, but to a mount point of a
26 * vehicle that contains one or more vehicle parts.
27 *
28 * It's supposed to be the basic vehicle interface for the @ref map.
29 * Another class shall be used to get a reference to a specific part of the
30 * vehicle.
31 *
32 * Note that it must be created with a valid vehicle reference and a valid
33 * part index. An instance can become invalid when the referenced vehicle is
34 * changed (parts added / removed or whole vehicle removed). There is no way
35 * to detect this (it behaves like C++ references).
36 */
