13 * These functions live here to provide various JSON-loaded entities

16 * To subscribe, a JSON-loaded entity simply needs to have a

18 * (As well as storing their id in an 'id' member of either string or int id type)

20 * If the entity is loaded with generic_factory, no further changes are needed.

39/** If those conditions are satisfied, keep track of where this item has been modified */
