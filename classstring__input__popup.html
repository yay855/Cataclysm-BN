Detailed Description

Shows a window querying the user for input.

Returns the input that was entered. If the user cancels the input (e.g. by pressing Escape), an empty string is returned. An empty string may also be returned when the user does not enter any text and confirms the input (by pressing ENTER).

Examples:

input = string_input_popup().title("Enter something").query();
// shows the input field in window w at coordinates (10,20) up to (30,20).
input = string_input_popup().window(w, 10, 20, 30).query();
