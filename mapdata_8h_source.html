208/*
209 * List of known flags, used in both terrain.json and furniture.json.
210 * TRANSPARENT - Players and monsters can see through/past it. Also sets ter_t.transparent
211 * FLAT - Player can build and move furniture on
212 * CONTAINER - Items on this square are hidden until looted by the player
213 * PLACE_ITEM - Valid terrain for place_item() to put items on
214 * DOOR - Can be opened (used for NPC pathfinding)
215 * FLAMMABLE - Can be lit on fire
216 * FLAMMABLE_HARD - Harder to light on fire, but still possible
217 * DIGGABLE - Digging monsters, seeding monsters, digging with shovel, etc
218 * LIQUID - Blocks movement, but isn't a wall (lava, water, etc)
219 * SWIMMABLE - Player and monsters can swim through it
220 * SHARP - May do minor damage to players/monsters passing through it
221 * ROUGH - May hurt the player's feet
222 * SEALED - Can't use 'e' to retrieve items, must smash open first
223 * NOITEM - Items 'fall off' this space
224 * NO_SIGHT - When on this tile sight is reduced to 1
225 * NO_SCENT - Scent on this tile (and thus scent diffusing through it) is reduced to 0. This acts like a wall for scent
226 * MOUNTABLE - Player can fire mounted weapons from here (e.g. M2 Browning)
227 * DESTROY_ITEM - Items that land here are destroyed
228 * GOES_DOWN - Can use '>' to go down a level
229 * GOES_UP - Can use '<' to go up a level
230 * CONSOLE - Used as a computer
231 * ALARMED - Sets off an alarm if smashed
232 * SUPPORTS_ROOF - Used as a boundary for roof construction
233 * MINEABLE - Able to broken with the jackhammer/pickaxe, but does not necessarily support a roof
234 * INDOORS - Has roof over it; blocks rain, sunlight, etc.
235 * COLLAPSES - Has a roof that can collapse
236 * FLAMMABLE_ASH - Burns to ash rather than rubble.
237 * REDUCE_SCENT - Reduces scent even more, only works if also bashable
238 * FIRE_CONTAINER - Stops fire from spreading (brazier, wood stove, etc)
239 * SUPPRESS_SMOKE - Prevents smoke from fires, used by ventilated wood stoves etc
240 * PLANT - A "furniture" that grows and fruits
241 * LIQUIDCONT - Furniture that contains liquid, allows for contents to be accessed in some checks even if SEALED
242 * OPENCLOSE_INSIDE - If it's a door (with an 'open' or 'close' field), it can only be opened or closed if you're inside.
243 * PERMEABLE - Allows gases to flow through unimpeded.
244 * RAMP - Higher z-levels can be accessed from this tile
245 * EASY_DECONSTRUCT - Player can deconstruct this without tools
246 * HIDE_PLACE - Creature on this tile can't be seen by other creature not standing on adjacent tiles
247 * BLOCK_WIND - This tile will partially block wind
248 * FLAT_SURF - Furniture or terrain or vehicle part with flat hard surface (ex. table, but not chair; tree stump, etc.).
249 *
250 * Currently only used for Fungal conversions
251 * WALL - This terrain is an upright obstacle
252 * THIN_OBSTACLE - This terrain is a thin obstacle, i.e. fence
253 * ORGANIC - This furniture is partly organic
254 * FLOWER - This furniture is a flower
255 * SHRUB - This terrain is a shrub
256 * TREE - This terrain is a tree
257 * HARVESTED - This terrain has been harvested so it won't bear any fruit
258 * YOUNG - This terrain is a young tree
259 * FUNGUS - Fungal covered
260 *
261 * Furniture only:
262 * BLOCKSDOOR - This will boost map terrain's resistance to bashing if str_*_blocked is set (see map_bash_info)
263 * WORKBENCH1/WORKBENCH2/WORKBENCH3 - This is an adequate/good/great workbench for crafting. Must be paired with a workbench iexamine.
264 */
