bool is_char_allowed ( int  ch)

Returns whether or not the given (ASCII) character is usable.

Called to check player name validity and world name validity. Only printable symbols not reserved by the filesystem are permitted.

Parameters
chThe char to check.
Returns
true if the char is allowed in a name, false if not.
