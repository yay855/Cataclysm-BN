Helper for place_player Flood-fills the area from a given point, then returns the area it found.

Considers unpassable tiles passable if they can be bashed down or opened. Will return INT_MAX if it reaches upstairs or map edge. We can't really use map::route here, because it expects a destination Maybe TODO: Allow "picking up" items or parts of bashable furniture and using them to help with bash attempts.
