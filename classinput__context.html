Detailed Description

Represents a context in which a set of actions can be performed.

This class is responsible for registering possible actions (traditionally keypresses), handling input, and yielding the correct action string descriptors for given input.

This turns this class into an abstraction method between actual input(keyboard, gamepad etc.) and game.
