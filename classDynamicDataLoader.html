Detailed Description

This class is used to load (and unload) the dynamic (and moddable) data from json files.

There exists only one instance of this class, which can be accessed with get_instance

Usage is basically this:

  • Let user decide which world to play in.
  • Call unload_data (to unload data from a previously loaded world, if any)
  • Call load_data_from_path(...) repeatedly with different paths for all the mods of the current world.
  • Call finalize_loaded_data when all mods have been loaded.
  • Play.

The object initializes itself upon first usage. It also unloads everything when the program ends.

Porting stuff to json works like this:

  • create a function void load_my_object(JsonObject &jo);
  • Add an entry to type_function_map (inside of initialize) that calls the new function.
  • Inside that function load the data from the json object. You must also provide a reset function and add a call to that function in unload_data
  • Optional: create a finalize function and call it from finalize_loaded_data
  • Optional: create a function to check the consistency of the loaded data and call this function from check_consistency
  • Than create json files.
